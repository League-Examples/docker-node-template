# Architectural Patterns

This directory contains reference documentation for features that are
not part of the template itself but are common patterns you may want to
add to applications built from the template. Each document describes the
pattern, its implementation approach, and references the inventory app
where a working implementation exists.

These are meant as guides for an AI agent or developer who wants to add
the feature to a specific application. They are not todos — they describe
how to do something, not a commitment to do it.

## Available Patterns

- **qr-codes.md** — QR code generation and short URL routing for
  physical objects or locations
- **search-service.md** — Global search across multiple entity types
  with unified results
- **slack-bot.md** — Slack bot integration for notifications and
  conversational AI
